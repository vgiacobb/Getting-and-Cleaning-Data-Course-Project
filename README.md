# Getting and Cleaning Data Course Project

## Files
- `run_analysis.R`: Script to download, merge, clean, and create the tidy dataset.
- `tidy_dataset.txt`: Final tidy dataset with averages for each subject and activity.

## How to run
1. Place all files in the same folder.
2. Run `run_analysis.R` in R or RStudio.
3. The tidy dataset `tidy_dataset.txt` will be generated.

## Description
This project processes data from Samsung Galaxy S smartphone to create a tidy dataset containing 
the average of mean and standard deviation features for each activity and subject.
